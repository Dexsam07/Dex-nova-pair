# Levanter Pairing Site Clone

यह एक WhatsApp Multi-Device pairing code generator website है।  
levanter.site जैसी दिखती और काम करती है।  
यहाँ से pairing code generate करके WhatsApp bot को link कर सकते हो।

## Features
- Phone number से pairing code generate
- Dark theme + clean UI (Tailwind + DaisyUI)
- Copy button
- Error handling
- Mobile responsive
